# 🏋️‍♂️ Gym Power - Landing Page High Conversion

Proyecto de landing page profesional diseñada específicamente para gimnasios, centros de fitness y entrenadores personales que buscan digitalizar su oferta y centralizar la información.

## 🚀 Características principales
- **Mobile First:** Diseño optimizado para celulares, donde se genera el 80% del tráfico de gimnasios.
- **Grilla de Horarios:** Sección interactiva para consulta rápida de clases.
- **Sección de Planes:** Visualización clara de membresías para acelerar la decisión de compra.
- **Contacto Directo:** Integración estratégica de puntos de contacto (WhatsApp/Formularios).

## 🛠️ Tech Stack
- **Framework:** [Vue.js 3] (Composition API)
- **Herramienta de Construcción:** [Vite]
- **Estilos:** CSS3 / Flexbox & Grid
- **Despliegue:** [Netlify](https://www.netlify.com/)

## 📦 Instalación y Desarrollo
1. Clonar el repo: `git clone https://github.com/Gabriel0511/LandingGymDemo.git`
2. Instalar dependencias: `npm install`
3. Correr en local: `npm run dev`
